#今日地球
--
日本的向日葵8号卫星每天都会拍摄大量的地球图片，这些图片还是比较震撼的，因此在PC端、移动端有很多软件可以实现几乎实时（10min）的桌面壁纸切换，本项目就来自己实现一个简单的壁纸切换APP。

##以实现功能
* 最新地球图片的获取
* 壁纸的设置
* 定时自动更新
